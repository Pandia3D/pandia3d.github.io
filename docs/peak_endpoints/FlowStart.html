FlowStart¶

GET /v1/startVolumeFlowMeasurement¶

Starts the volume flow measurement. Note that startVolumeMeasurement should be called beforehand.

Example request:

GET /v1/startVolumeFlowMeasurement HTTP/1.1
Host: 192.168.2.103:5001
Accept: text/html

Example response:

HTTP/1.1 200 OK
Access-Control-Allow-Origin: *
Content-Length: 4
Content-Type: text/html; charset=utf-8
Date: Mon, 25 Jul 2022 11:07:46 GMT
Server: waitress

{OK}

Example response-content:

{OK}
{Failed due to network error}
{Failed due to incomplete system setup}
Status Codes:

  • 200 OK – No error

  • 404 Not Found – Error
